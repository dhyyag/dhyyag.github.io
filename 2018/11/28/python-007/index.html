python——命名规范

一、模块名

  • 小写
  • 避免使用下划线

    二、类名

  • 首字母大写，驼峰命名规则
  • 私有类用一个下划线开头

    三、函数名

  • 小写，多个单词用下划线分割
  • 私有函数用一个下划线开头

    四、变量名

  • 小写，多个单词用下划线分割

    五、常量名

  • 全部大写，多个单词用下划线分割
