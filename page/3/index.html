jmeter-查看结果树

查看结果树

查看结果树仅用于功能测试或测试计划调试和验证期间。
在负载测试期间不要使用视图结果树，因为它会消耗大量资源(内存和CPU)。

可以通过“查看结果树”查看request，响应所需的时间，以及响应数据（可以选择数据的显示格式，比如：Text，JSON，HTML 等），也可以查看日志（点击页面右上方的叹号，可展示日志，也可关闭日志），见下图红色标记处。

2.元组()

元组是不可变的数据类型。

元组最通常的用法是用在打印句子中:

3.字典 {}

字典类似于你通过联系人名字查找地址和联系人详细情况的地址簿，把键（名字，是唯一的）和值（详细情况）练习在一起。
键：使用不可变的对象（比如字符串）作为字典的键
值：使用不可变的或可变的对象作为字典的值
d = {key1 : value1 , key2 : value2}

4. 序列 （列表、元组和字符串都是序列）

序列的两个主要特点是索引操作符和切片操作符
索引操作符：可以让我们从序列中抓取一个特定的项目
切片操作符：可以让我们能够获取序列的一个切片，即一部分序列
第一个数（冒号之前）表示切片开始的位置，第二个数（冒号之后）表示切片到哪里结束

python——模块

若想再其他函数中重用很多函数，则可以使用模块，模块是一个包含了所有你定义的函数和变量的文件，模块的文件名必须以.py为扩展名。

1.使用标准库模块（sys模块）

import sys

from sys import *
但是为了避免名称冲突，尽量使用import语句

2.若我只想在程序本身被使用的时候运行主块，而在它被别的模块输入的时候不运行主块可以通过模块的name属性完成

3.使用dir()函数列出模块定义的标识符。（函数，类，变量）
