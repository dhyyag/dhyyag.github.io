python-字符串总结（一）

1. 单引号 ‘’

1.1 被两个单引号‘’括住的所有空格或制表符都保留。

1.2 单引号中可以使用双引号。

2. 双引号 ""

2.1 被双引号“”括住的所有空或格制表符都保留。

2.2 双引号中可以使用单引号。

3. 三引号''' '''

被三引号''' '''括住可以指示一个多行的字符串，也可以在三引号中使用单引号和双引号。

4. 转义符 \

4.1 用法一：转义符\可以转义单引号

4.2 用法二：在字符串中行末加一个反斜杠\表示字符串在下一行继续，而不是重新开始一个新的行。

5. 自然字符串 r R

若想指示某些不需要如转义符那样的特别处理的字符串时，则需要指定一个自然字符串。
注：处理正则表达式时，一定要用自然字符串

6. unicode字符串 u U

6.1 若字符串中有非英文文本时，比如中文，则需要指定unicode字符串。
