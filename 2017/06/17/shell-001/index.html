shell-网页下载wget

wget是一个用于文件下载的命令行工具。

1. 指定下载位置

wget URL
wget URL1 URL2

2. 指定输出位置

wget URL -O download -o log
-O ：指定输出的文件名
-o ：指定一个日志文件

3. 指定重试次数

wget -t 5 URL
-t ：指定重试次数
5 ：重试5次

4. 对下载限速

wget –limit-rate 20k URL
–limit-rate ：对wget限速
20k ：20年字节为指定速度限制

5. 断点续传

wget -c URL
-c ：从断点开始继续下载

6. 补充

wget –mirror URL
wget –user username –password pass URL
